# Choosing an Activity Source

Geo Activity Playground can get your activities from different places. Pick the path that matches your situation:

| Your situation                                                                                  | What to do                                        |
| ----------------------------------------------------------------------------------------------- | ------------------------------------------------- |
| You record with a Garmin, OsmAnd, OpenTracks, or similar and have files on your computer        | [Import activity files](import-activity-files.md) |
| You want to keep using Strava and sync from it automatically, with or without a source checkout | [Connect Strava API](connect-strava-api.md)       |
| You run the app on a server and want to upload files via the browser                            | [Upload activity files](upload-activity-files.md) |
| You want to leave Strava entirely and migrate your history                                      | [Moving from Strava](moving-from-strava.md)       |

⚠️ If you use the Strava API as your source, do **not** also copy raw files from a Strava archive into your `Activities/` folder. The two paths handle data differently and will produce duplicates or import errors. If you want to leave Strava, follow [Moving from Strava](moving-from-strava.md) instead.
